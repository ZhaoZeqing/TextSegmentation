# TextSegmentation

一个基于JAHMM的简单的中文分词器。

需要：[jahmm-0.6.1.jar](https://code.google.com/p/jahmm)

语料：[msr_training.utf8](http://sighan.cs.uchicago.edu/bakeoff2005/)


基本是按照这篇文章中的代码来写的，略做修改
